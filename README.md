# Pep parser

Консольный парсер документации Python и PEP, написанный на Python с использованием библиотеки **BeautifulSoup**. Скрипт собирает и анализирует данные с официальных страниц документации и PEP, а результаты выводит в терминал или сохраняет в файл.

## Возможности

Парсер поддерживает несколько режимов работы:

- `whats-new` — собирает ссылки на статьи о нововведениях в Python, переходит по ним и извлекает информацию об авторах и редакторах статей.
- `latest-versions` — собирает информацию о версиях Python и их статусах (stable, security-fixes, EOL и т.п.).
- `download` — скачивает архив с актуальной документацией Python.
- `pep` — собирает данные о всех документах PEP, сравнивает статусы в общем списке и на странице каждого PEP, считает количество документов по каждому статусу и общее количество PEP.

Результаты могут выводиться в разных форматах:
- построчный вывод в консоль;  
- табличный вывод в консоль;  
- сохранение в CSV‑файл.

## Технологии

Проект использует:

- `Python 3.9+`  
- `beautifulsoup4` — парсинг HTML  
- `requests` / `requests-cache` — HTTP‑запросы и кэширование ответов  
- `argparse` — парсинг аргументов командной строки  
- `prettytable` — табличный вывод в терминал  
- `csv` — запись результатов в файлы  
- `logging` — логирование работы и ошибок


## Установка и запуск

1. Клонировать репозиторий:

   ```bash
   git clone https://github.com/ssavboy/parser_pep.git
   cd pep_parser
   ```

2. Создать и активировать виртуальное окружение:

   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # Linux/macOS
   source venv/bin/activate
   ```

3. Установить зависимости:

   ```bash
   python -m pip install --upgrade pip
   pip install -r requirements.txt
   ```

4. Посмотреть справку по режимам:

   ```bash
   python main.py --help
   # или
   python main.py -h
   ```

## Примеры использования

Список доступных команд:

- Сбор новостей о нововведениях в Python:

  ```bash
  python main.py whats-new
  ```

- Информация о статусах версий Python с красивой таблицей:

  ```bash
  python main.py latest-versions -o pretty
  ```

- Скачивание архива с актуальной документацией:

  ```bash
  python main.py download
  ```

- Статистика по PEP с выводом в CSV‑файл:

  ```bash
  python main.py pep -o file
  ```

Опции могут включать, например:

- `-o {pretty,file}` / `--output {pretty,file}` — формат вывода (таблица в консоль или файл CSV).  
- `-c` / `--clear-cache` — очистка кэша запросов (если реализовано).

## Логи и отладка

Скрипт логирует ключевые этапы работы и ошибки через стандартный модуль `logging`. Уровень логирования и формат сообщений можно настроить в файлах проекта.
